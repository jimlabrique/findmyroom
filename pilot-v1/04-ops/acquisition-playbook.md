# Acquisition Playbook (Brussels first)

## Priority order

1. Existing coloc supply in Facebook groups
2. Candidate pull only after qualified room inventory exists

## Target channels

- Facebook groups (Brussels colocation / housing)
- Personal network referrals
- WhatsApp group referrals

## Qualification checklist for a coloc lead

- [ ] Room really available within next 45 days
- [ ] Contact responds within 24h
- [ ] Willing to complete form and use shortlist process
- [ ] Accepts pilot communication over WhatsApp
- [ ] Accepts structured compatibility screening

If all checked -> set status to `coloc_qualified`.

## Daily outreach target

- 20 outreach messages/day
- 5 live conversations/day
- 2 qualified colocs/day average in week 1 and 2

## Messaging rule

- Keep outreach under 6 lines
- No promises of guaranteed fill
- Position value as "better tri quality", not "more leads"
